Project : End to end ecommerce website testing using Amazon. (*Work in progress)

Technology used: Java 17, Selenium, TestNG, Maven, IntelliJ IDEA, Jenkins, Allure Reports, Extent Reports.

This project is to test Amazon ecommerce website end to end using automation testing.  

Test Scenarios:

1. Homepage (search, sort, filter, etc).
2. Cart (Adding product, price, removing product, billing etc).    
3. Login functionality (User login, user logout, correct credentials, wrong credentials, etc).

Implemented Jenkins and GitHub webhooks in this project for triggering testcases on new commits, using LocalTunnel to expose the Jenkins url running on local machine, soon be deployed on VM.

Implemented Allure reports as well as Extent reports.
