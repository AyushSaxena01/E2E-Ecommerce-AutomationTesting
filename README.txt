Project : End to end ecommerce website testing on Amazon.

Technology used: Java 17, Selenium, TestNG, Maven.

This project is to test Amazon ecommerce website end to end using automation testing.  

Test flow:

1. Homepage (search, sort, filter, etc).
2. Cart (Adding product, price, removing product, etc).    
3. Login functionality (User login, user logout, correct credentials, wrong credentials, etc).
